# kiro v1.2

ダウンロードフォルダに届いたファイルを、自動でプロジェクトフォルダへ振り分けるアプリです。

## インストール

### Mac

1. `kiro.app` をダウンロードする
2. `アプリケーション` フォルダへドラッグ＆ドロップする
3. 初回起動時に「開発元を確認できません」と表示された場合：
   - `kiro.app` を **右クリック → 開く** → 「開く」ボタンをクリック
   - 2回目以降はダブルクリックで起動できます

### Windows

1. `kiro.exe` をダウンロードする
2. 任意のフォルダに置く（例：`C:\Users\ユーザー名\kiro\`）
3. ダブルクリックで起動する
4. 「WindowsによってPCが保護されました」と表示された場合：
   - **詳細情報** をクリック → **実行** をクリック

> **SmartScreen 警告について**  
> kiro はコード署名証明書を取得していないため、初回起動時に Windows SmartScreen の警告が表示されます。  
> これは署名なしのアプリすべてに表示される仕組みであり、kiro 自体に問題があるわけではありません。  
> ソースコードは [GitHub で公開](https://github.com/Schues/kiro) しており、内容を確認いただけます。  
> 完全に警告を非表示にするにはコード署名証明書（有料）の取得が必要なため、現時点では未対応です。

## 起動・停止

アプリを起動すると、画面右下（Windows）またはメニューバー右上（Mac）にアイコンが表示されます。

| アイコン | 状態 |
|---------|------|
| 🟢 緑の丸 | 監視中 |
| ⚫ グレーの丸 | 停止中 |

### アイコンを右クリック（Mac は左クリック）するとメニューが開きます

```
● 監視中
──────────
停止する
設定を開く
──────────
終了
```

- **停止する / 開始する** … 監視のオン・オフを切り替えます
- **設定を開く** … プロジェクトフォルダや監視フォルダを変更します
- **終了** … アプリを完全に終了します

---

## 初回セットアップ：プロジェクトフォルダを登録する

1. トレイアイコンを右クリック → **設定を開く**
2. 「プロジェクトを追加 / 編集」欄で以下を入力する
   - **名前**：好きな名前（例：`仕事`, `請求書`, `写真`）
   - **パス**：振り分け先のフォルダ（**参照…** ボタンで選択できます）
3. **追加 / 更新** をクリックする
4. 必要な数だけ繰り返す
5. **閉じる** をクリックして完了

> 監視フォルダはデフォルトで `~/Downloads`（ダウンロードフォルダ）です。  
> 変更する場合は設定画面下部の「監視フォルダ設定」から変更してください。

## 使い方

1. アプリが起動（緑アイコン）している状態でブラウザなどからファイルをダウンロードする
2. ダウンロード完了後、自動でポップアップが表示される

   ```
   新しいファイルを検知しました
   report_2024.pdf
   
   移動先プロジェクトを選択してください
   ┌─────────────┐
   │ 仕事        │
   │ 請求書      │
   │ 写真        │
   └─────────────┘
   [ 振り分ける ]  [ 振り分けしない ]  [ ⚙ 設定 ]
   ```

3. 移動先を選んで **振り分ける** をクリックすると、ファイルが自動で移動される
4. 今回は振り分けなくてよい場合は **振り分けしない** をクリックする

ファイルは `プロジェクトフォルダ / YYYY-MM-DD /` という日付フォルダの中に保存されます。

## 操作ログ

振り分けの記録は `move_log.csv` に自動保存されます。

- Mac：`kiro.app` と同じフォルダ
- Windows：`kiro.exe` と同じフォルダ

Excel などで開いて確認できます。

---

## プロジェクトフォルダを直接編集する（上級者向け）

設定画面を使わず `projects.json` をテキストエディタで直接編集することもできます。

```json
{
  "仕事": "/Users/yourname/Documents/仕事",
  "請求書": "/Users/yourname/Documents/請求書",
  "写真": "C:\\Users\\yourname\\Pictures"
}
```

- パスは絶対パスで記載してください
- フォルダが存在しない場合は振り分け時にエラーになります

## よくある質問

**Q. アプリを起動してもアイコンが見当たらない**
- Windows：画面右下のタスクバーにある `∧` をクリックすると隠れたアイコンが表示されます
- Mac：メニューバー右上（時計の左あたり）を確認してください

**Q. 起動時に「WindowsによってPCが保護されました」と毎回表示される**
- 初回のみ表示されます。**詳細情報 → 実行** をクリックしてください。2回目以降は表示されません。
- この警告はコード署名証明書（有料）がないアプリすべてに表示される Windows の仕様です。kiro 固有の問題ではありません。

**Q. 「すでに起動中のインスタンスがあります」と表示される**
- kiro はすでに起動中です。タスクバー / メニューバーのアイコンを確認してください
- アイコンが見当たらない場合は、PC を再起動してから再度起動してください

**Q. ポップアップが出ない**
- トレイアイコンが緑になっているか確認してください（グレーの場合は「開始する」を選択）
- 監視フォルダが正しく設定されているか確認してください（設定 → 監視フォルダ設定）

**Q. 誤って振り分けてしまった**
- `move_log.csv` で移動先を確認し、手動でファイルを戻してください

**Q. Windows 起動時に自動でアプリを立ち上げたい**
- `kiro.exe` のショートカットを作成し、以下のフォルダに入れてください：
  `C:\Users\ユーザー名\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup`

**Q. Mac 起動時に自動でアプリを立ち上げたい**
- システム設定 → 一般 → ログイン項目 → `+` ボタンで `kiro.app` を追加してください
